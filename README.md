# 🚨 Smart IoT Security Alert System

## 📖 Overview

This project is a Smart IoT Security Alert System that uses an **ESP32-CAM**, **GSM module**, **PIR motion sensor**, **buzzer**, and **LED indicators** to provide real-time security monitoring and instant alerts.

When motion or intrusion is detected, the system automatically captures an image, activates local alarms, sends an SMS, makes a phone call, and emails the captured photo to the authorized user.

---

## ✨ Features

* 📸 Capture images using ESP32-CAM
* 🚶 Detect motion with PIR sensor
* 📱 Send SMS alerts via GSM
* 📞 Make emergency phone calls
* 📧 Send email notifications with captured images
* 🔊 Activate buzzer alarms
* 💡 LED status indication
* ⏱️ Real-time intrusion detection and alerts

---

## 🛠️ Components

* ESP32-CAM
* GSM Module (SIM800L/SIM900)
* PIR Motion Sensor
* Buzzer
* LEDs
* Power Supply

---

## 🔄 System Workflow

1. Motion is detected by the PIR sensor.
2. ESP32-CAM captures an image.
3. Buzzer and LED alert are activated.
4. GSM module sends an SMS and places a phone call.
5. Captured image is sent via email.
6. User receives instant notifications.
